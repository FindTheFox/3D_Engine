# Doom-Nukem
Project game Doom-Nukem like, for 42 school Paris

## Built
  C language, SDL2 library

## Utilisation

*       <strong>Player</strong>
* Movement:   WASD
* Camera:   ARROW / MOUSE
* Up:   SPACE
* Down:   RCTRL
* Sprint:   LSHIFT

*       <strong>OBJ Movement</strong>
* Change selected obj:  PNUM_0
* Change Mode (Lock 0 / Rotation 1 / Movement 2:
                        PNUM_ENTER
* Action: PNUM_[1-9]
